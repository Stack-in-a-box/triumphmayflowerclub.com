# Security Policy

## Supported versions

The only version being actively supported is the version that is currently deployed to production, i.e. the latest stable SemVer version.

| Version        | Supported          |
|:--------------:|:------------------:|
| `0.x`          | :x:                |
| `1.0.x`        | :x:                |
| `1.1.x`        | :x:                |
| `1.2.<latest>` | :white_check_mark: |
| `2.0.0-alpha`  | :x:                |
